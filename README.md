# ProjetFlask
Projet Flask en groupe


# Installation local use
On your computer:

Install the python environment:
```sh
python3 -m venv venv

#Activation
source venv/bin/activate
# For Fish : source venv/bin/activate.fish

pip install -r ./requirements.txt
```

# loancer le server
On your computer:

```sh
flask --app app run
```
